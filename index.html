Getting started

Kevlr is a responsive front-end framework based on Sass. This framework is not made to built prototypes looking all the same but it's made to built real websites with their own visual identity.

Get the code on Github

Beta

For now Kevlr is still in Beta, all the features aren't here yet and maybe you'll encounter some bug. If that happend please let us know

Sass

This framework uses Sass, if you don't know how to use it, please visit their website. You don't need to understand everything but if you get how works the variables and mixins, it will be easier.

Variables

The most important file in Kevlr is the 'variables.scss' file. That's where you can change every styling rules like how many columns you want in your grid system, the width between the columns, the font sizes...

Mobile first

In Kevlr we built everything for mobile first and then the orther screen sizes because we think mobile is the most complex part of the design.
