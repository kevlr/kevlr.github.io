Grid System

Flex Grid

One of the particularity in Kevlr is the grid system. Indeed this one is built with 'display:flex;'. This allows you to stop worrying about clearing floats*, re-order elements*, have equal height content without javascript...

Base grid

Variables

The first step to built your grid system is to change the variables according to your design.

$columns-number: 12;		// Number of columns in your grid system.
$column-gutter: 10px;		// Width of the gutter between the columns.
$container-max-width: none;	// The max-width of your container, "none" if fluid.
$container-min-width: 320px;	// The min-width of your container
$container-align: center;	// Position of your container, "center", "left" or "right",
