Grid System

Flex Grid

One of the particularity in Kevlr is the grid system. Indeed this one is built with 'display:flex;'. This allows you to stop worrying about clearing floats*, re-order elements*, have equal height content without javascript...

Base grid

Variables

The first step to built your grid system is to change the variables according to your design.

$columns-number: 12;		// Number of columns in your grid system.
$column-gutter: 10px;		// Width of the gutter between the columns.
$container-max-width: none;	// The max-width of your container, "none" if fluid.
$container-min-width: 320px;	// The min-width of your container
$container-align: center;	// Position of your container, "center", "left" or "right"

Responsiveness

Variables

Choose the number of screen sizes you want adding/removing one value to each parameter.

// layouts
$larger-layouts-name:			's',		'm',		'l';	// name of you layouts (ascending order)
$larger-layouts-media-queries:		500px,		800px,		1024px;	// your break points for each layout
$larger-layouts-columns-number:		12,		12,		12;	// your columns number for each layout
$larger-layouts-column-gutter:		20px,		20px,		40px;	// gutter size for each layout
$larger-layouts-container-max-width:	480px,		768px,		960px; 	// container max-width for each layout ("none" if fluid)

// general
$media-queries-rules: 'only screen';
$elements-to-order: -3, 3; // order propriety for flex display

Syntax

Sizes

To change the columns width in another screen size add a .width# class followed by -- and the name of your layout, for exemple .width6--m

Exemple
.width12.width6--m
.width12.width6--m
.width12.width6--m
.width12.width6--m
<div class="row">
  <div class="column width12 width6--m">...</div>
  <div class="column width12 width6--m">...</div>
  <div class="column width12 width6--m">...</div>
  <div class="column width12 width6--m">...</div>
</div>

Alignment

Change the column alignment adding .bottom-align, .center-align, .baseline-align or .equal-height class to your .row
