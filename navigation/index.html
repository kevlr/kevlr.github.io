Navigation

Navigation types

The kevlr navigation can be displayed in 3 different ways: vertically, horizontally and horizontally with equal width for each element. Simply add a .nav-vertcal, .nav-horizontal or .nav-equal-width to your .nav element.

By default, you have 2 different styles for the navigation: .nav-primary and .nav-secondary. To style them go to the variables.scss file.

Example

html
<ul class="nav nav-equal-width nav-primary">
  <li><a>...</a></li>
  <li><a>...</a></li>
  <li><a>...</a></li>
  <li><a>...</a></li>
</ul>
