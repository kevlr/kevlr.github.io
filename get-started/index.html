Get started

Requirements

Sass 3.3+

Kevlr uses Sass, if you don't know it, please visit their website. You don't need to understand everything but if you get how works the variables and mixins, it will be easier.

Kevlr needs the 3.3 Sass version or greater because it uses "maps".

How to install

  1. Download the repository from Github
  2. Fill the variables.scss file
  3. Import the kevlr.scss in your main scss file
  4. Start writing your custom rules

Browser support

Kevlr is compatible with all modern browsers : Chrome 21+, Firefox 28+, Safari 6.1+, IE10+, Opera 12.1+. But we still have fallback for all older browsers except IE7 and lower.
