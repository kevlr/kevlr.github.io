Typography

Variables

Fill the variables in the variables.scss file to change the font-sizes, color, line height... For the font sizes, enter the value in pixels, we will transform them in rem and put a fallback in pixels for older browsers. For the line height, we recommend to use integer values.

Headings

In kevlr, we think headings are separated in 2 kinds: those you use for your layout (for example, the title of your website is, usually, in a h1) and those that compose the main content as articles. So in the variables.scss you have the main heading style and then the heading placed in a container we called .article.

Useful mixins

rem

We recommend to use rem for the font size. To do it easily we created a mixin to convert pixels in rem.

scss file
.element {
  @include rem(font-size, 14px);
}
css output
.element {
  font-size: 14px
  font-size: 0.875rem;
}

Responsive font

We created a mixin to directly enter responsive font sizes, this mixin is called responsive-font-size, as a value enter a bracket with the values you want for each screen size separeted by a comma.

Example

scss file
.element {
  @include responsive-font-size((12px, 14px, 16px));
}
css output
.element {
  font-size: 12px
  font-size: 0.75rem;
}
@media only screen and (min-width: 600px) {
  .element {
    font-size: 14px;
    font-size: 0.875rem;
  }
}

@media only screen and (min-width: 1000px) {
  .element {
    font-size: 16px;
    font-size: 1rem;
  }
}
