Typography

Units

Kevlr use rem unit for the types with a fallback in pixels for the oldest browsers

Variables

Responsive Types

Use the variables to make the types responsive. for the $body-font-size put a value for each layout (in ascending order) separated by a comma. For each value, you can add a second value, it will be the line-height. for the headings font-size, it will be the same execpt you can put a ratio instead of the exact pixel size. The ratio will be based on the $body-font-size.

$body-font-size: 	18px, 		20px, 		20px, 		22px;
$h1-size: 		48px 1.1,	52px, 		52px 1,		58px;
$h2-size: 		7/3 1.1;
$h3-size: 		6/3 1.1;
$h4-size: 		5/3 1.2;
$h5-size: 		4/3 1.2;
$h6-size: 		3/3 1.3;
